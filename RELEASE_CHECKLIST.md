# ICC DSL Release Checklist

Use this checklist before publishing a language release.

- Confirm the target channel: stable, preview, or experimental.
- Update parser, catalog, and version metadata together.
- Add migration notes for changed syntax.
- Run the public language test suite.
- Verify ICC-GO consumes the same language release.
- Verify icc-dsl.org and ICC-GO docs cite the same stable version.
